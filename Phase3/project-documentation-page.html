Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM).
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server.

JavaScript and Java
  • Purpose:
    • Java: A general-purpose, object-oriented programming language used for developing a wide range of applications, including:
      • Android apps
      • Enterprise software
      • Desktop applications
      • Web servers
    • JavaScript: A scripting language primarily used for adding interactivity and dynamic behavior to web pages. It's also increasingly used for:
      • Building web applications (with frameworks like React, Angular, Vue)
      • Server-side development (with Node.js)
      • Game development
  • Execution:
    • Java: Compiled language, meaning the code is translated into bytecode before execution by the Java Virtual Machine (JVM).
    • JavaScript: Interpreted language, meaning the code is executed directly by the browser or other runtime environment without prior compilation.
  • Typing:
    • Java: Statically typed, meaning the data types of variables must be declared explicitly and checked at compile time.
      • // Declaring a variable of type int int age = 25;
      // Declaring a variable of type String String name = "John Doe";
      // Declaring a variable of type boolean boolean isStudent = true;
    • JavaScript: Dynamically typed, meaning the data types of variables are determined at runtime and can change during execution.
      • // Declaring a variable using the 'var' keyword var age = 25;
      // Declaring a variable using the 'let' keyword let name = "John Doe";
      // Declaring a variable using the 'const' keyword (for constants) const PI = 3.14159;
  • Syntax:
    • Java: Syntax is similar to C++ and other traditional object-oriented languages.
    • JavaScript: Syntax is influenced by C, but also has features like first-class functions and prototypal inheritance.
  • Ecosystem:
    • Java: Robust ecosystem with a vast collection of libraries, frameworks, and tools.
    • JavaScript: Vibrant and rapidly evolving ecosystem with a focus on web development.

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).

var x = 42; let y = 'Hello'; const Z = true;

You can use most of ISO 8859-1 or Unicode letters such as å and ü in identifiers. You can also use the Unicode escape sequences as characters in identifiers.

Some examples of legal names are Number_hits, temp99, and _name.
